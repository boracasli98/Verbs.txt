<h1>Welcome to the Verbs.txt project!</h1>
<p>This is a collaborative project of contributing verb conjugations in various languages.</p>

<h1>How can I help?</h1>
<p>You can help by creating and improving templates, adding new verbs and spotting mistakes in the verb conjugations, but you need to fork the repository at first. By the way, almost all languages lack full conjugation templates right now.</p>
